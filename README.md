# RASP4PHP

Runtime Application Self-Protection for PHP, based on [Frida](https://www.frida.re). (DEMO)


# Installation

```bash
# sudo sysctl -w kernel.yama.ptrace_scope=0
# sudo sysctl -p
$ pipenv install
```
